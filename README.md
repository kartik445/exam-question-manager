# Exam Question Manager

## Description
The Exam Question Manager is a web application that allows users to create, manage, and distribute exam questions efficiently. It offers a user-friendly interface and supports various question types.

## Features
- Create and edit questions
- Categorize questions by subjects
- Generate random exam papers
- User management
- Secure access and data management

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/kartik445/exam-question-manager.git
   cd exam-question-manager
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```
Open your browser and navigate to `http://localhost:3000`.

## Deployment Instructions
To deploy the application:
1. Build the application:
   ```bash
   npm run build
   ```
2. Deploy to your preferred hosting service (e.g., Heroku, AWS).
3. Ensure to set up the environment variables for production.
